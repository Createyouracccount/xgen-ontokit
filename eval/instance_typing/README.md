# 인스턴스 세밀 타이핑 — 실측 후 폐기 기록 (2026-07-14)

가설: NER 인스턴스(TTA 15 대분류에만 rdf:type)를 형태소 접미 일치 세밀 클래스로
재타입(서울고등학교: 기관→고등학교)하면 GraphRAG 의 `클래스→subClassOf*→인스턴스`
leg 공회전이 풀린다.

## 폐기 사유 (심판 판정 APPROVE)

- 구조적 원인: Kiwi 사전화 + NER 표면형 대부분이 비합성 고유명 → 조정으로 오를 여지 없음.
- 0.16% 에 정밀도 불안 → 본체 코드 제거 (무효과 기능 배포 금지, CLAUDE.md 단순성 원칙).
- ⚠️ 후과: ABox→TBox 연결 기제 부재 확정 → 추출 클래스의 "인스턴스 보유" completeness 는
  구조적으로 0 → completeness 개선은 분모 축소(클래스 승격 게이트)로만 가능.

NER 동일명 강등 조건으로 승계. `retyped.json` = v1 실측 원본.
